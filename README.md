# 🎓 Student Registration Form

A simple and beginner-friendly Student Registration Form built using **HTML5**. This project demonstrates the use of various HTML form elements such as text inputs, radio buttons, checkboxes, dropdown menus, text areas, and submit buttons.

## 📌 Project Description

The Student Registration Form collects basic student information including:

- Name
- Phone Number
- Gender
- City
- Hobbies
- Email Address
- Additional Notes

This project is ideal for beginners who are learning HTML forms and user input handling.which helps them to start their beginning.

## 🚀 Features

✅ Student Name Input Field  
✅ Phone Number Input Field  
✅ Gender Selection (Radio Buttons)  
✅ City Selection (Dropdown Menu)  
✅ Hobbies Selection (Checkboxes)  
✅ Email Input Field  
✅ Notes Section (Textarea)  
✅ Submit Button

## 🛠️ Technologies Used

- HTML5
